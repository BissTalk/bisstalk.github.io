It is a period of static and boring web experiences. Rebel developers and marketers armed with Sitecore have decided to take matters into their own hands and stage sneak attack on the evils of static web sites and win people over with dynamic and personalized experiences.

During the battle to win over customers with awesome experiences, the Rebels used xDB to listen to their customers and tell Sitecore how to self tune itself using machine learning.

Developers quickly realized that awesome experiences should not be confined to only web pages so they expanded their mission to native applications using Sitecore Item Web API, emails using the Email Experience Manager, and the used FXM for revitalizing legacy static web sites...
